# 🎓 Beyond Enrollment: Student Success & Dropout Risk Dashboard

### 📊 Analytics Tools and Techniques | 2026  
Developed by **Balictar, Biñas, and Soleño**

---

## 📌 Overview

Behind every student record is a story shaped by academic performance, financial capacity, and personal circumstances. While some students successfully graduate, others fall behind or leave the system.

This dashboard explores patterns in student data to identify key factors associated with **dropout risk and academic success**. By analyzing academic, demographic, and socio-economic indicators, the system provides insights that can support **early intervention and improved student retention strategies**.

---

## 📂 Dataset Information

This project uses the *Predict Students’ Dropout and Academic Success* dataset, developed to support early identification of at-risk students using machine learning techniques.

- 🎯 **Objective:** Predict student outcomes (*Dropout, Enrolled, Graduate*)
- 🧑‍🎓 **Unit of Analysis:** Individual student records
- 📊 **Features:** Academic performance, demographics, socio-economic factors
- 🏛 **Funding:** SATDAP Program – Capacitação da Administração Pública (Portugal)

The dataset underwent preprocessing to ensure quality, including handling anomalies and outliers.

---

## 🚀 Features of the Dashboard

- 📈 **Key Performance Indicators (KPIs)**  
  - Total Students  
  - Dropout Rate  
  - Graduation Rate  

- 📊 **Interactive Visualizations**
  - Admission Grade Distribution (Box Plot)
  - Age Distribution (Violin Plot)
  - Student Outcome Distribution (Donut Chart)
  - Dropout Rate by Course and Gender (Heatmap)
  - Dropout Contribution by Course (Treemap)

- 🎛 **Interactive Filters**
  - Course selection  
  - Gender filtering  
  - Sorting options
